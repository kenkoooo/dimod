Decorators¶

Decorators that provide input checking and format for Ising and QUBO functions and methods.

Decorators¶

qubo(Q_arg) Provides input checking for quadratic unconstrained binary optimization (QUBO) methods.
ising(h_arg, J_arg) Provides input checking for Ising methods.
qubo_index_labels(Q_arg) Replaces the labels in the dictionary of QUBO coefficients with integers.
ising_index_labels(h_arg, J_arg) Replaces the variable labels in h and J with integers.
