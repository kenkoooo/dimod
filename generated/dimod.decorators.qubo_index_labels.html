dimod.decorators.qubo_index_labels¶

qubo_index_labels(Q_arg)[source]¶

Replaces the labels in the dictionary of QUBO coefficients with integers.

Parameters:Q_arg (int) – Location of the Q argument in args.
Returns:Function that replaces the user-given Q with one with integer labels, runs the decorated method, then restores the previous labels.

Notes

The integer labels start with 0.

If the given labels are orderable, the relabeling is applied to the nodes sorted lexicographically.

Examples: Decorate functions like this:

@qubo_index_labels(1)
def sample_qubo(self, Q):
    pass

@qubo_index_labels(0)
def qubo_energy(Q, sample):
    pass
