dimod.responses.SpinResponse.as_binary¶

SpinResponse.as_binary(offset=0, data_copy=False)[source]¶

Converts a SpinResponse to a BinaryResponse.

Parameters:
  • offset (float/int, optional) – The energy offset as would be returned by ising_to_qubo. The energy offset is applied to each energy in the response.
  • data_copy (bool, optional) – Whether to create a copy of each data dict. Default False.
Returns:

A BinaryResponse with the samples converted from spin to binary, the energies updated with offset and all of the data transferred directly.
Return type:

BinaryResponse

Notes

Only information stored in data property and as would be returned by samples(data=True) is transferred.
